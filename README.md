使用说明(要增加或替换的脚本在 src 目录下)
--------

1. 添加三个脚本
   * ＺPBZ全拼输入法引擎
   * ＺPBZ键盘
   * ＺPBZ绘图函数
   **注意**：【ＺPBZ全拼输入法引擎】必须在【ＺPBZ键盘】前面
2. 替换一个脚本
   * Scene_Name
